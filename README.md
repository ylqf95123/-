# 百度网盘文件夹搜索器

一个轻量的浏览器插件，帮助你快速搜索并定位百度网盘中的文件夹。

## 功能特性

- 🔍 **快速搜索** - 输入关键词，快速找到目标文件夹
- ⌨️ **键盘导航** - 支持上下键选择，Enter 进入
- 📁 **自动展开** - 点击结果自动导航到目标文件夹
- ⌨️ **快捷键** - `Ctrl + K` 快速呼出搜索面板
- 🎨 **美观界面** - 现代化悬浮面板设计

## 安装步骤

### 1. 下载插件

将 `baidu-folder-search` 文件夹下载到本地。

### 2. 加载到浏览器

**Edge / Chrome:**

1. 打开扩展管理页面
   - Edge: `edge://extensions/`
   - Chrome: `chrome://extensions/`

2. 开启右上角的「开发者模式」

3. 点击「加载解包的扩展」

4. 选择 `baidu-folder-search` 文件夹

### 3. 使用插件

1. 打开 [百度网盘](https://pan.baidu.com) 并登录
2. 进入任意文件夹
3. 按 `Ctrl + K` 或点击右下角 🔍 按钮打开搜索面板
4. 输入文件夹名称关键词
5. 使用上下键选择，或直接点击结果
6. 自动导航到目标文件夹

## 工作原理

```
用户输入关键词
     ↓
触发网盘页面搜索功能
     ↓
解析搜索结果中的文件夹
     ↓
自定义面板展示结果
     ↓
用户点击 → 自动导航
```

## 注意事项

1. **登录状态** - 使用前请确保已登录百度网盘
2. **首次加载** - 首次使用时建议刷新网盘页面
3. **选择器适配** - 如搜索结果为空，可能是网盘页面更新导致选择器变化

## 常见问题

### Q: 搜索不到结果怎么办？

A: 可能的原因：
- 网盘页面结构更新，选择器需要调整
- 尝试刷新页面后重试
- 检查是否已登录

### Q: 如何自定义快捷键？

A: 目前快捷键固定为 `Ctrl+K`，后续版本将支持自定义。
